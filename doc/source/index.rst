PyGen - Documentation
=====================

`PyGen` a versatile and extensible file generator.

Installation
------------

`PyGen` operates as a command-line tool, `pygen` provided via Python entry points.

You may install it by using `pip`::

   pip install pygen

The ``pygen`` command will then be available to you, provided that the Python's
scripts directory is in your ``PATH``.
